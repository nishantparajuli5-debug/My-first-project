# 🔐 Login Form Project

A clean and modern **Login Form UI** built using **HTML and CSS**.
This project demonstrates best practices in structuring a simple authentication interface with a responsive and user-friendly design.

---

## 📌 Features

* ✨ Simple and elegant UI design
* 📱 Responsive layout (mobile-friendly base)
* 🎨 Modern gradient background
* 🔒 Password input field with proper type
* ✅ Required field validation (HTML level)
* 🔁 Ready for backend integration

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

```
login-project/

```

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/login-project.git
```

### 2. Navigate to project folder

```bash
cd login-project
```

### 3. Open in browser

Simply open `index.html` in your browser.
